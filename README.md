# UpcomingMCU API

UpcomingMCU (UMCU) API is a free, consumable-only REST API to retrieve data about the productions within the [Marvel
Cinematic Universe (MCU)](https://en.wikipedia.org/wiki/Marvel_Cinematic_Universe).
